# Delhi Metro Analysis Projec

* A complete end-to-end data analysis project exploring ridership trends, metro line performance, revenue, and passenger behavior using Delhi Metro trip data (150,000+ records).
* This project demonstrates data cleaning, feature engineering, EDA, financial analysis, visualization, and actionable insights.

✅ Dataset Summary

* 150,000+ rows, 10 raw columns

* Key fields: Date, From_Station, To_Station, Fare, Passengers, Ticket_Type

* Missing values handled: Ticket_Type → Mode, Passengers → Median
  
* Cleaned station names & mapped them to Metro Lines

✅ Work Done

* Data cleaning & preprocessing

* Feature engineering: Day_Name, Month_Name, Year

* Metro_Line

* Revenue & Trip_Profit

* Passenger trend analysis (daily, monthly, yearly)

* Metro line performance analysis

* Ticket type usage patterns

* Revenue & profit evaluation

✅ Key Insights

* Yellow Line is the busiest corridor

* Smart Cards dominate usage (regular commuters)

* Weekdays show higher traffic than weekends

* Profit varies by line due to cost differences

* Blue & Yellow contribute maximum revenue
